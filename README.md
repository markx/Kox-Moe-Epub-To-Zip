## Usage
### Download executable from [Release](https://github.com/bennyz327/Kox-Moe-Epub-To-Zip/releases/latest)
#### Command Line Arguments
| arg_name       | default  |
|------------|----------|
| epub_file  | ./       |
| output_dir | ./output |
```shell
epub_to_zip(.exe) [epub_file] [output_dir]
```
